​# Customer Behaviour Analysis Project

## Overview
This project provides a comprehensive analysis of customer purchasing patterns and demographic trends. By integrating database querying, statistical programming, and interactive visualization, the study identifies key drivers of customer engagement and revenue growth.

## Tech Stack
* **SQL:** Data extraction, joining complex tables, and preliminary data structuring.
* **Python:** Data cleaning, exploratory data analysis (EDA), and predictive modeling using Pandas and Seaborn.
* **Power BI:** Interactive dashboarding and KPI tracking.
* **Excel:** Initial data auditing and documentation.

## Project Structure
* **`DA CUSTOMER PROJECT SQL.sql`**: Contains the backend logic for data retrieval and transformation.
* **`data analysis project.ipynb`**: The primary analytical environment showing the step-by-step cleaning and visualization process.
* **`Report-Customer Behaviour Analysis.pdf`**: Final executive summary detailing strategic recommendations based on the data findings.
* **`New Microsoft Excel Worksheet (2).xlsx`**: The source dataset utilized throughout the pipeline.
* **`Screenshots/`**: Visual previews of the analytical dashboards.

## Key Performance Indicators
* **Customer Retention:** Analysis of repeat purchase rates and long-term loyalty.
* **Revenue Distribution:** Identification of high-value customer segments and regional performance.
* **Behavioral Trends:** Correlation between demographic factors and product preferences.

## Usage
1. Open the **SQL** script to review the database schema and query logic.
2. Run the **Jupyter Notebook** to view the statistical distributions and data correlations.
3. Reference the **PDF Report** for a high-level summary of the business insights.
